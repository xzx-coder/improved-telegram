## 一、项目目标

设计并实现一个能够绕过现代网站反爬检测的智能爬虫系统，重点解决：
1. **TLS指纹伪装**（JA3/JA4）—— 让爬虫在TLS握手层"看起来像"真实浏览器
2. **浏览器指纹伪造**（Canvas/WebGL/AudioContext）—— 让爬虫的浏览器环境无法被识别
3. **自动化特征隐藏**（WebDriver/CDP）—— 消除Playwright/Puppeteer的自动化痕迹
4. **人类行为模拟**（鼠标轨迹/键盘节奏/滚动模式）—— 让操作行为不像机器

最终交付一个**完整的反检测爬虫系统 + Web可视化分析平台 + 课程报告**。

## 二、技术选型

| 类别 | 选型 | 理由 |
|------|------|------|
| 浏览器自动化 | Playwright (Python) | 比Selenium更现代，CDP协议更接近真实浏览器 |
| 指纹伪装 | playwright-stealth + 自研JS注入 | 业界成熟的反检测方案 |
| HTTP客户端 | curl_cffi | 支持JA3指纹伪装，处理纯HTTP请求 |

| 后端框架 | Flask | 轻量，适合课程作业规模 |
| 前端可视化 | ECharts + Bootstrap | 多种图表展示 |
| 测试工具 | bot.sannysoft.com / browserleaks.com | 验证伪装效果 |
| 目标网站 | Cloudflare演示站 / 知乎 / 淘宝 | 真实环境测试 |
| 编程语言 | Python 3.9+ | 生态丰富 |

## 四、核心模块详细设计

### 模块1：TLS指纹伪装模块
- 使用 `curl_cffi` 替代 `requests`，模拟Chrome的JA3指纹
- 关键参数：cipher_suites、extensions、elliptic_curves、ec_point_formats
- 配置文件：`config/tls_profiles.json`（保存Chrome 120、Firefox 121等真实指纹）

### 模块2：浏览器指纹伪造模块
- Canvas指纹：注入随机噪声到 `toDataURL` 输出
- WebGL指纹：伪造GPU渲染器字符串（如 "ANGLE (NVIDIA GeForce)"）
- AudioContext指纹：注入随机扰动到音频采样
- 字体指纹：返回固定的常见字体列表
- 屏幕分辨率：使用真实常见分辨率（如 1920x1080）
- WebRTC：禁用本地IP泄露
- 实现方式：每个指纹维度写一个独立的JS注入脚本

### 模块3：自动化特征隐藏模块
- 删除 `navigator.webdriver` 属性
- 修复 `navigator.plugins` 和 `navigator.languages`
- 隐藏Chrome DevTools Protocol特征（`cdc_`开头的变量）
- 修改 `Permissions` API行为
- 修复 `chrome.runtime` 缺失问题
- 使用 `playwright-stealth` 作为基础 + 自研补丁

### 模块4：人类行为模拟模块
- **鼠标轨迹**：使用贝塞尔曲线生成自然路径
  - 起点到终点之间插入3-5个控制点
  - 添加随机抖动和高斯噪声
  - 不同时刻使用不同的加速度曲线
- **键盘输入**：模拟真人击键间隔
  - 平均间隔 80-250ms，标准差 50ms
  - 偶尔出现停顿（500-1500ms）
  - 错字回退修正
- **页面滚动**：模拟阅读行为
  - 不规则滚动速度
  - 随机回滚
  - 停留时间与内容长度相关
- **页面停留**：访问后等待 2-8 秒再点击

### 模块5：代理管理模块
- 代理池：定时从代理网站抓取免费代理 + 付费API
- 健康检查：自动剔除失效代理
- 智能切换：根据目标网站自动选择代理地域
- 失败重试：单次失败自动切换代理重试 3 次

### 模块6：调度与控制模块
- 任务队列：使用 Redis 或内存队列
- 并发控制：信号量限制同时运行的浏览器数量
- 异常处理：网络异常、验证码、IP封禁的应对策略
- 日志系统：详细记录每次请求的指纹、代理、响应

### 模块7：数据存储模块
- MongoDB 集合设计：
  - `crawl_results`：爬取结果
  - `fingerprint_records`：指纹数据
  - `detection_logs`：反检测评估日志
  - `proxy_pool`：代理池

### 模块8：可视化分析模块
- 仪表盘：总爬取数、成功率、平均响应时间
- 对比图表：不同伪装策略的有效性对比
- 指纹雷达图：伪装前后各项指纹差异
- 趋势图：检测率随时间变化
- 热力图：人类行为模拟的真实度分析

### 示例1：贝塞尔曲线鼠标轨迹生成器
```python
import random
import numpy as np

def bezier_mouse_path(start, end, num_points=20):
    """生成自然的鼠标轨迹"""
    sx, sy = start
    ex, ey = end
    
    # 生成2-3个随机控制点
    control_points = [(sx, sy)]
    for i in range(2):
        cx = sx + (ex - sx) * (i + 1) / 3 + random.randint(-50, 50)
        cy = sy + (ey - sy) * (i + 1) / 3 + random.randint(-50, 50)
        control_points.append((cx, cy))
    control_points.append((ex, ey))
    
    # 三次贝塞尔曲线
    t = np.linspace(0, 1, num_points)
    points = []
    for ti in t:
        x = (1-ti)**3 * control_points[0][0] + \
            3*(1-ti)**2*ti * control_points[1][0] + \
            3*(1-ti)*ti**2 * control_points[2][0] + \
            ti**3 * control_points[3][0]
        y = (1-ti)**3 * control_points[0][1] + \
            3*(1-ti)**2*ti * control_points[1][1] + \
            3*(1-ti)*ti**2 * control_points[2][1] + \
            ti**3 * control_points[3][1]
        points.append((int(x), int(y)))
    
    return points
```

### 示例2：Playwright 反检测初始化
```python
from playwright.sync_api import sync_playwright
from playwright_stealth import stealth_sync

def create_stealth_browser(proxy=None, fingerprint=None):
    with sync_playwright() as p:
        browser = p.chromium.launch(
            headless=False,
            proxy={"server": proxy} if proxy else None,
            args=['--disable-blink-features=AutomationControlled']
        )
        context = browser.new_context(
            viewport={'width': 1920, 'height': 1080},
            user_agent='Mozilla/5.0 (Windows NT 10.0; Win64; x64) ...'
        )
        page = context.new_page()
        stealth_sync(page)
        # 注入自定义指纹脚本
        page.add_init_script(open('scripts/canvas.js').read())
        page.add_init_script(open('scripts/webgl.js').read())
        return browser, context, page
```

// scripts/canvas.js
const originalToDataURL = HTMLCanvasElement.prototype.toDataURL;
HTMLCanvasElement.prototype.toDataURL = function(type) {
    const result = originalToDataURL.apply(this, arguments);
    // 添加随机噪声
    const noise = Math.random().toString(36).substring(2, 8);
    return result.replace(/^data:image\/png;base64,/, 
                          'data:image/png;base64,') + noise;
};

## 八、风险评估与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 反爬技术升级 | 实验结果失效 | 选择多个目标网站对比验证 |
| 法律合规问题 | 可能违反目标网站ToS | 仅爬取公开数据，不进行DDoS等攻击 |
| 时间不足 | 功能无法完成 | 优先实现核心3个模块（指纹、行为、检测绕过） |
| 浏览器版本差异 | 指纹不匹配 | 使用真实浏览器抓取的最新TLS配置文件 |
| IP被封 | 测试无法进行 | 实现代理池，付费代理作为备选 |
